# Changelog

## 1.0.0 - 2026-03-06

Initial public release.

Highlights:

- check in to nearby points of interest with one tap
- nearby POI search via MapKit with configurable category filtering
- YAML preview with embedded map before check-in
- check-in history with search and date filtering
- JSON and CSV export of check-in history
- JSON import with deduplication
- SwiftData local persistence with legacy JSON migration
- debug mode for diagnostics
